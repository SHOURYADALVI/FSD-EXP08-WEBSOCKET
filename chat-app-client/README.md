# Chat App Client

This is a minimal React (Vite) frontend for the Socket.IO chat server.

Run server:
- cd chat-app-server
- npm install
- npm run dev

Run client:
- cd chat-app-client
- npm install
- npm run dev

Open http://localhost:5173 in two tabs. Messages from you appear on the right; others on the left.

Config:
- Server URL is http://localhost:3001 (see src/App.jsx)
- Server CORS should allow http://localhost:5173
